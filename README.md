# Decidely-Next-JS
Frontend application for Decidely App. Backend on https://github.com/ComradeStijn/Decidely-API.

## Status
### Finished
- Login Page
- Voting Page
- Admin Panel


## Getting Started
### Install Instruction
1. Clone the repo
2. Install NPM packages and build
  ```sh
  npm install --lregacy-peer-deps && next build
  ```
3. Start server
  ```
  npm start
  ```
